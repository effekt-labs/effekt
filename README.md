## Features

- Free & Open Source
- Ultra-lightweight & Powerful
- Modular & Fully Tree-Shakeable
- Framework-Independent
- Written in TypeScript
- Native SSR Support
- No External Dependencies
- Extremely Easy to Use
- API-Friendly
