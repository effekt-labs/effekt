## Features

- Free & Open-source
- Lightweight & Powerful
- Supports ESM, IIFE & UMD
- Framework independent
- Written in TypeScript
- Test-covered codebase
- Fully tree-shakeable
- SSR-ready
- No dependencies
- Super easy to use
- API friendly
